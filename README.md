## Installation

To integrate WeMap_ARM64 into your Xcode project using CocoaPods, specify it in your `Podfile`:

```ruby
platform :ios, '14.0'
use_frameworks!

target 'YourApp' do
  pod 'WeMap_ARM64', :git => 'https://github.com/your-username/WeMap_ARM64.git'
end
